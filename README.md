# Baritone

Baritone is a pathfinding and navigation mod for Minecraft focused on precise, reliable movement through the world. It allows players to automatically travel to locations, coordinates, and waypoints while intelligently handling terrain, obstacles, and basic interactions.

## Features

- Advanced pathfinding across terrain, caves, and structures  
- Automatic movement including walking, jumping, climbing, and mining  
- Waypoint system for saving and navigating to important locations  
- Real time route recalculation when the environment changes  
- Highly configurable behavior for different playstyles  

## Waypoints

Baritone supports waypoints that can be created, named, and removed in game. Pathfinding goals can directly target saved waypoints, making long distance travel and navigation more convenient and consistent.

## Usage

Baritone is controlled through in game commands and keybinds. You can set goals, manage waypoints, and adjust settings without leaving the game. Refer to the command list or configuration files for detailed control options.

## Installation

1. Install a compatible Minecraft mod loader  
2. Download the Baritone mod jar  
3. Place the jar in your `mods` folder  
4. Launch the game  

## Compatibility

- Designed for singleplayer and multiplayer environments  
- Works best with standard survival mechanics  
- Server side installation is not required unless restricted by server rules
